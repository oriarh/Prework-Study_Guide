HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head element contains metadata, that helps browser render the web app

CSS

  • CSS is used for styling the web page
  • There are three ways to style through CSS.
    1. Inline css
    2. Internal Style Sheet
    3. External Style Sheet
  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside the element
  • We can style HTML using elements as selectors or by using class attribute as a selector

GIT

  • Git is a version control software
  • Github is a version control software that is on the cloud
  • Git Status checks what branch we are on
  • git checkout -b "enter branch name" creates a new branch and takes us to it
  • We use the following steps to push add, commit and push our work upto github from our local branch
    1. git add -A; to push our code to the staging area
    2. git commit -m "Our Message" - to commit the changes to the feature branch
    3. git pull origin main - to make sure our local branch is up to date before pushing in the changes, with the most update main branch in github
    4. git push feature/add-js - to push our branch up for commit

JAVASCRIPT

  • JavaScript is an object oriented programming language OOP
  • A variable is a name container that allows us to store data in our code
  • Variables have to be defined before we can use them
  • Control flow is the order in which a computer executes the code
